# Dataset Information

## Overview
This directory contains the datasets used for medical insurance cost prediction.

## Files
- insurance.csv: Main dataset file containing insurance information
- synthetic_data.csv: Generated synthetic data for testing (if needed)

## Data Description
The main dataset (insurance.csv) contains the following columns:
- age: Age of primary beneficiary
- sex: Insurance contractor gender
- bmi: Body mass index
- children: Number of children covered by insurance
- smoker: Smoking status
- region: Beneficiary's residential area
- charges: Individual medical costs billed by insurance

## Usage
1. Place your dataset file (insurance.csv) in this directory
2. Use the data_preprocessing.py module to load and process the data

## Notes
- The .gitkeep file ensures this directory is tracked by git
- Large data files should be added to .gitignore
